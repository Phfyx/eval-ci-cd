# 📦 Items API

Une API REST simple pour gérer une collection d'items. Elle permet d'obtenir la liste des items et d'en créer de nouveaux.

## 🚀 Fonctionnalités

- 🔍 Récupérer tous les items
- ➕ Ajouter un nouvel item
